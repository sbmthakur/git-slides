# WHAT IS GIT?
---
# Simple Git commands
---
```jsx
$ git init
$ git status
$ git add <file>
$ git rm <file>
$ git commit [-m]
$ git branch
$ git push [origin] [master]
$ git pull [origin] [master]
```
---
# Usual workflow
```jsx
$ git pull [origin] [master]
$ git add/rm [file]
$ git commit [-m] [--no-verify]
$ git push [origin] [master]
```
---
# Slightly Advanced Git commands
---
```jsx
$ git log
$ git show
$ git diff [branch_1] [branch_2]
$ git checkout [branch]
$ git fetch [origin]
$ git merge [origin/master] 
$ git stash
```
---
# Advanced/Lethal Git commands
```jsx
$ git rebase --> Rewrites history
$ git cherry-pick --> Rewrites history
$ git checkout -f --> Can delete code
$ git reset --hard [anything] --> Can delete code
$ git stash drop --> Can delete code
```
